#LinkIt jQuery Plugin#

**LinkIt** is  simple jQuery plugin to attach links to elements

##Version##
0.1.0

##Usage##
    $("span").linkIt({
        href: "http://yahoo.com",
        text: "Click Me",
        target: "_blank"
    });

##Vendors##
jQuery [http://jquery.com](http://jquery.com)
